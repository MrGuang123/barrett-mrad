
## 项目介绍

Barrett MRAD(Multi-RoleAdaptivDesign)：巴雷特多用途自行调应设计
练习后端服务，后面会越来越充实

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
